# Municipal Road Maintenance Standard Operating Procedures

## Priority Classification
- **P1 (Critical)**: Risk score > 75 OR fatal accident reported. Response within 2 hours.
- **P2 (Warning)**: Risk score 56-75 OR multiple major accident reports. Response within 8 hours.
- **P3 (Elevated)**: Risk score 31-55 OR rising trend detected. Response within 24 hours.
- **P4 (Normal)**: Risk score < 31. Scheduled maintenance cycle.

## Repair Actions
1. **Pothole**: Temporary cold-mix patch within response window. Permanent hot-mix within 72 hours.
2. **Drainage Block**: Deploy jetting machine + manual clearing. Inspect adjacent segments.
3. **Road Crack**: Seal with bitumen emulsion if <5mm. Full resurfacing if >5mm.
4. **Debris**: Clear with JCB + manual labor. Issue fine if construction-related.
5. **Waterlogging**: Activate pump if >6 inches. Deploy barricades at entry points.

## Resource Allocation
- Each ward has 2 repair crews available per shift.
- Emergency crew can be diverted if P1 alert is issued.
- Heavy machinery (JCB, roller) requires 4-hour mobilization.

## Escalation Protocol
- If 3+ segments in same zone reach Warning: escalate to Zonal Commissioner.
- If any segment reaches Critical for >4 hours: escalate to Municipal Commissioner.
- Fatal accident on degraded segment: automatic inquiry + media response preparation.
